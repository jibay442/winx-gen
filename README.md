## 🚀 Lancer le projet

### Prérequis

- [Docker](https://www.docker.com/) + Docker Compose installés

### Démarrage

```bash
docker compose up --build
```

Puis ouvre **http://localhost:5173** dans ton navigateur.

---

## 🛑 Arrêter

```bash
docker compose down
```

Pour tout supprimer (base de données incluse) :

```bash
docker compose down -v
```
